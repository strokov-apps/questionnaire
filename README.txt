Install Node.js (need to run install and run Grunt)
after that install command line interface to use Grunt (need for static resources processing like minification of js files)
>npm install -g grunt-cli

package.json - used to define Grunt dependencies(tasks)
gruntfile.js - used to define how use tasks

Then install actually Grunt with all related tasks
>npm install

Install Ruby (need for scss, don't forget to mark checkbox "Add to PATH".)
after that get latest updates for RubyGems
>gem update --system

Install Compass
>gem install compass

Install Sass
>gem install sass
